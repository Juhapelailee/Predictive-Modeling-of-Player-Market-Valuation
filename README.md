# Predictive-Modeling-of-Player-Market-Valuation

This project aims to build a data-driven player valuation model that predicts the estimated market value of players using a variety of features such as position, age, height, weight, nationality, team, performance metrics, and league characteristics.
To run the app with Streamlit:
Type following in the terminal:
git clone https://github.com/Juhapelailee/Predictive-Modeling-of-Player-Market-Valuation/
streamlit run app.py
